# 영양 데이터 출처로 식약처 가공식품 표준데이터 채택

NutriRank의 영양 데이터 출처로 식약처/공공데이터포털의 「전국통합식품영양성분정보(가공식품) 표준데이터」를 채택한다. 브랜드 상품 단위이고, 100g/100ml 기준량과 당류·포화지방·나트륨·식이섬유 등 등급 산출에 필요한 필드를 갖췄으며 무료·공개이기 때문이다.

## Considered Options

- **Open Food Facts** — 바코드 연계에 유리하나 국내 제품 커버리지가 약함
- **제조사/쇼핑몰 직접 크롤링** — 최신·다양하나 유지보수·법적 리스크가 큼
- **수동 큐레이션** — 품질은 최고지만 확장성이 없음(초기 보완용으로만 병행)
